HTML Semantic Elements

In HTML5 semantic elements were introduced. These are tags that are more descriptive of the content they enclose. Prior to HTML5 only div and span tage were available to mark different sections of our page and though they were effective, the introduction of semantic tags made it easier to identify different sections of our page based on their purpose or function. In-Page Hyperlinking

in-page hyperlinking allows us to provide user friendly navigation for our lengthier pages. This is achieved by tagging the desired element with an anchor tag (<a>) but instead of an href attribute, we set an ID attribute in the <a> tag. This ID is then referenced from our hyperlink.
