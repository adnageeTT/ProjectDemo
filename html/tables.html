Merging adjacent table cells

In HTML tables it is possible to merge adjacent cells setting the rowspan and colspan attributes. Colspan is used to indicate how many column widths in our table we want the merger cell to occupy, while rowspan indicates the number of rows we want to merge together. It is important to note that when we span on a cell (<td> or <th>) we would need to provide fewer td or th tags for that row as shown in the examples below.
